How to run the Program from the Terminal

1- Open terminal and navigate to the project root folder inTeger-reading-to-sorted-linkedList
2- Run the following command with no quotation " javac -d . implementation/*.java "
3 - Run the following command with no quotation to run the program " java implementation.Main "

------------------------------------------------

How to create the documentation for the program 
1- Open terminal and navigate to the project root folder inTeger-reading-to-sorted-linkedList
2- javadoc -d doc implementation/*.java
3- navigate to doc folder and open the index.html file in a browser
